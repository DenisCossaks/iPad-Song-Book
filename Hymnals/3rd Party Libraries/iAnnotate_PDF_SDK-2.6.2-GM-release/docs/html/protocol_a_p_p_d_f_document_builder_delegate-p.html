forPDFAtPath: (NSString *)  pdfPath 
[optional]

Used when the document builder requires a password to access or modify the PDF file.

Parameters:
builderThe document builder object.
passwordTypeThe type of password required. Most often this will be kAPPDFPasswordTypeUser for the user password, although in some circumstances the owner password may be required.
pdfPathThe path of the PDF document for which the password is required. OR...
pdfDataNSData containing the PDF document for which the password is required.
Returns:
The requested password, or nil to abort builder.

shouldCancelAfterProgress: (CGFloat)  progress 
[optional]

Used when the builder reaches a checkpoint during an export operation to indicate progress and give the delegate an opportunity to cancel the opeartion. A cancelled export will return NO and set its error field appropriately.

Parameters:
progressA value between 0 and 1 that specified how much of the export operation has completed.
Returns:
YES is the operation should be cancelled/aborted; NO if it should continue.
