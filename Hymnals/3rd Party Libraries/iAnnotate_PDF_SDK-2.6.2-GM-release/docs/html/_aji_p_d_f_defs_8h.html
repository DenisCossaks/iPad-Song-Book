enum APPDFWriteOptions

Options controlling PDF generation behavior.

Enumerator:
kAPPDFWriteOptionsCopy 

The default write options. Makes a copy of the PDF, including all annotations.

kAPPDFWriteOptionsCopyOriginal 

Same as kAPPDFWriteOptionsCopy. Only retained for backwards compatibility, use kAPPDFWriteOptionsCopy.

kAPPDFWriteOptionsFlatten 

"Flattens" all annotations into the page content. Extra pages may be generated and added to the document in order to accomodate annotation text. Note that this will embed the annotations directly into the document, so that they will not be editable or removable.

kAPPDFWriteOptionsAnnotatedPagesOnly 

Specifies that only annotated pages should be written.

kAPPDFWriteOptionsStripUserAnnotations 

Specifies that all (user-editable) annotations should be stripped, leaving only the "original" PDF content.

kAPPDFWriteOptionsUsePageRange 

Indicates the the "pageRange" parameter should be used to control which pages of the PDF should be generated.

kAPPDFWriteOptionsStripDeadObjects 

Creates a new version of a "cluttered" PDF with any unused ("residual") objects removed. The resulting file should appear exactly the same in all PDF viewers. If an error occurs, there may still be a partially-generated file for the client to clean up.

This operation may be time consuming, especially for large PDFs on older iPads.

If this option is specified, only kAPPDFWriteOptionsStripUserAnnotations may also be specified.
