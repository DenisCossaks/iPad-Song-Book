- (UIView *) masterViewForPDFController: (APPDFViewController *)  controller [optional]

Used to obtain the "master" view in which this PDF view is embedded, which should be a top-level full-screen application view. This is used, for example, when displaying an image annotation in full-screen mode.

If this method is not implemented or 'nil' is returned, then the UIWindow's rootViewController property is used to obtain the master view. So if your application has set the rootViewController, there is no need to implement this method.
