pathForAnnotationType: (APAnnotationType)  annotType 
[optional]

Used to provide a path for annotation types for which a file might be provided, such as file attachments and custom stamp images. Note that the file will be read into device memory and ultimately embedded in the PDF file upon a writeback/sync event. Thus, care must be taken with the size of specified files. (No size checking will be done by the library.)

Returns:
The absolute path to the file to use.
